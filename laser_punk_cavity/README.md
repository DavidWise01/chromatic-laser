# Six Point Coherence Cavity — Laser Tensor Prototype

A local-only informational webpage and simulation for the six-point laser tensor model.

Aesthetic direction: punk-zine rawness plus sterile technical minimalism.

## Files

- `index.html` — full webpage and animated prototype
- `cavity.kernel` — distilled model rules

## Safety boundary

This is not a physical laser construction guide. It contains no power supply instructions, optics sourcing list, cavity alignment procedure, or gas pressure recipe. It is a conceptual simulation of optical-cavity behavior: containment, gain, phase alignment, and output coupling.
