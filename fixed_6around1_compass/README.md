# Fixed 6 Around 1 Compass Array

A local-only symbolic optical/phase simulator.

- Four compass stations: North, East, South, West
- Each station has six fixed satellites around one local carrier
- No rotation or spinning geometry
- The active station fires into the central zero well
- Each station advances phase by 0.25
- Pulse form: `27 -> 00 -> 27`

Open `index.html` in a browser.
