# Chromatic 17 Laser Cavity

A local browser simulation of a 17-point chromatic coherence cavity: sixteen colored emitters arranged as four quadrants around a seventeenth pure-white synthesis point.

This is a conceptual optical-cavity visualization, not a physical laser build guide.

## Files

- `index.html` — interactive visual showcase
- `chromatic17.kernel` — compact kernel model
