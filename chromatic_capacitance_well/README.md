# Chromatic 17 Capacitance Well

A local-only visual simulation of sixteen fixed chromatic emitters firing into a central pure-white capacitance well.

This is an optical-cavity concept model, not a physical laser construction guide.

## Files

- `index.html` — interactive visual prototype
- `chromatic17_capacitance.kernel` — symbolic kernel/spec

## Model

- 16 fixed emitters
- 4 quadrants
- 4 emitters per quadrant
- Point 17 center well
- 98% containment / 2% gated release
