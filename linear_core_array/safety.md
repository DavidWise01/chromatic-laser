# Safety Notes

This package is a low-power reference/simulation. Do not use it as a guide for high-power lasers.

Minimum real-world controls:
- enclosed optical path
- appropriate eyewear for wavelength and power
- beam stop
- interlock
- emergency power cutoff
- current-limited drivers
- no reflective jewelry/tools near the path
- never aim at people, vehicles, aircraft, animals, or unknown surfaces
